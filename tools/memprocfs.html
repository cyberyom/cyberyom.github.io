Installing MemProcFS

MemProcFS can be found on Ulf Frisk’s github page. Ulf is a Swedish IT security specialist who currently owns a company called MemSec AB. MemProcFS is as simple as download the zipped up binaries on the released page of Ulf’s GitHub. As of the time of writing this, MemProcFS is currently on version 5.6.3.


Once it is downloaded, you can simple run it via CLI using MemProcFS.exe and you will be greeted with the help menu.
